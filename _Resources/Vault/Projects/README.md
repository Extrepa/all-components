## Project Structure

### Code Projects (Required)
**Location:** `/Users/extrepa/Projects/project-name/`

Each code project includes:
- README.md - Project overview
- INDEX.md - Project index
- PROJECT_STATUS.md - Current status
- docs/index.md - Documentation index
- docs/architecture.md - Architecture (if complex)
- docs/project-structure.md - File structure (if organized)
- docs/DEVELOPER_GUIDE.md - Developer guide (recommended)

**Templates:** `_Resources/_Templates/Documentation/`

### Project Management (Optional)
**Location:** `_Resources/Vault/Projects/Project-Name/`

If using separate project management docs, each project includes:
- README.md - Project management overview
- specs.md - Technical specifications
- tasks.md - Task tracking
- versions.md - Version history
- prompts.md - AI prompts used
- build-order.md - Build sequence
- assets.md - Asset documentation

**Templates:** `_Resources/Vault/Templates/Projects/`
