## Quick Start
The gallery is a static HTML file with thumbnails. Just serve it:
```bash
cd single-file-projects/ai-studio
# Using Python
python3 -m http.server 8000

# Using Node.js http-server
npx http-server -p 8000

# Using PHP
php -S localhost:8000
```
