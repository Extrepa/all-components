# UI Asset Integration Guide

**Last Updated**: December 10, 2025  
**Asset Type**: UI Assets (Icons, Sprites, Fonts)

## Overview

This guide provides instructions for integrating UI assets into the Errl Club Simulator project.

## Size Guidelines

- **Icons**: SVG preferred, PNG fallback, < 50KB
- **Sprites**: PNG with transparency, < 200KB
- **Fonts**: WOFF2 format, < 100KB per weight

## Integration Steps

1. Place asset in `/public/ui/{category}/{name}.{ext}`
2. Use SVG for scalable icons
3. Create sprite sheets for multiple icons
4. Load via CSS or JavaScript
5. Update asset catalog

## Best Practices

- Use SVG for icons (scalable)
- Create sprite sheets for multiple icons
- Use WOFF2 for fonts
- Optimize file sizes
- Document in asset catalog
