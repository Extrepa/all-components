## Asset IDs Generated

All 380 variant files have been assigned asset IDs following the pattern:
- `ERRL_VARIANT_<NAME>_<NUMBER>_<VARIANT_NUM>`
- Example: `ERRL_VARIANT_DYNAMIC_30_01`, `ERRL_VARIANT_DYNAMIC_30_02`, etc.

These asset IDs are registered in the asset registry and can be used throughout the application.
