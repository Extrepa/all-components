# Errl Character - 25 SVG Variations

## Overview
This collection contains 25 unique color variations of the Errl character, all using the same base SVG path data but with different color schemes.

## What's Included
- **25 SVG files** - Each variation is a complete, standalone SVG file
- **5 Color Categories** - Purple/Violet, Green, Red/Pink, Orange, and Blue
- **Consistent Structure** - All files use the same paths with different styling

## File Structure
Each SVG contains these elements:
- **Body** - Main character body shape
- **Face** - Facial area
- **Eyes (Left & Right)** - Two separate eye shapes
- **Mouth** - Mouth/expression shape
- **Outline** - Character outline/border

## Usage
These SVG files can be:
- Displayed directly in web browsers
- Embedded in HTML/CSS
- Imported into graphic design software (Adobe Illustrator, Figma, Sketch, etc.)
- Scaled to any size without quality loss
- Modified/customized further as needed

## Naming Convention
Files are named as: `errl_XX_name.svg`
- `XX` = Two-digit number (01-25)
- `name` = Color scheme name in lowercase with underscores

Example: `errl_01_classic_purple.svg`

## Color Scheme Structure
Each variation includes:
- **Body Color**: Main character body
- **Face Color**: Face/mask area (usually lighter)
- **Eyes Color**: Eye fill color (usually darker)
- **Mouth Color**: Mouth fill color (matches eyes)
- **Outline Color**: Stroke/border color (lighter, matching theme)
