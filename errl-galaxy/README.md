# Errl Galaxy

A Next.js application for exploring and visualizing galaxy structures using Three.js and React Three Fiber.

## Overview

Errl Galaxy is a 3D galaxy visualization application built with Next.js, React, and Three.js. It provides interactive exploration of galaxy structures and cosmic visualizations.

## Features

- 3D galaxy visualization
- Interactive exploration
- React Three Fiber integration
- Lamina shader materials
- Zustand state management

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```
