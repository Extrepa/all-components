## File Organization

### Unity Project Structure

**Assets/ (4MB)**
- **Scenes**: Game scenes (GameScene.unity, SampleScene.unity)
- **Scripts**: C# scripts (7,576 files in _Scripts/)
- **Sprites**: Sprite assets and prefabs
  - RainbowSlug.prefab - Main character
  - chunk_square.prefab - Chunk/square prefab
- **Prefabs**: Prefab assets
- **Settings**: URP and render pipeline settings
- **Input Actions**: InputSystem_Actions.inputactions

**Library/ (1.9GB - Should be gitignored)**
- Unity-generated cache files
- PackageCache: Downloaded packages (1.4GB)
- Bee: Build cache (146MB)
- Artifacts: Compiled assets (133MB)
- BurstCache: Burst compiler cache (105MB)
- Other Unity build artifacts

**ProjectSettings/**
- Unity project configuration (26 files)
- ProjectVersion.txt: Unity 6000.2.10f1
- TagManager.asset, ProjectSettings.asset, etc.

**Packages/**
- manifest.json: Package dependencies
- packages-lock.json: Locked package versions

### Documentation

- **Root**: INDEX.md, PROJECT_STATUS.md, README.md
- **docs/**: Architecture, project structure, completion checklist

## Key Directories

### Assets/
Primary game assets directory:
- **Scenes**: Game levels and scenes
- **Scripts**: C# game logic (7,576 files)
- **Sprites**: Visual assets and prefabs
- **Settings**: Render pipeline configuration

### Library/
Unity-generated cache (should be gitignored):
- Regenerated by Unity on project open
- Contains compiled assets, package cache, build artifacts
- **Size**: 1.9GB (mostly PackageCache at 1.4GB)

### ProjectSettings/
Unity project configuration:
- Render pipeline settings
- Input system configuration
- Physics settings
- Build settings

## File Naming Conventions

- **Scenes**: `*.unity` files
- **Scripts**: `*.cs` files (C#)
- **Prefabs**: `*.prefab` files
- **Meta Files**: `*.meta` files (13,944 files - Unity-generated)
- **Assets**: Organized by type in `_Scenes/`, `_Scripts/`, `_Sprites/`, etc.

## Unity-Specific Files

### .meta Files
- Unity generates .meta files for all assets
- Contains import settings and GUIDs
- **Count**: 13,944 .meta files
- **Purpose**: Asset tracking and import configuration

### ignore.conf
- Plastic SCM ignore configuration
- Defines what to exclude from version control

### .plastic/
- Plastic SCM version control data
- Workspace configuration
- Change tracking

## Build Output

Unity builds create platform-specific executables:
- Build location configured in Build Settings
- Output format depends on target platform
- No build artifacts stored in repository

## Important Notes

### Library/ Folder
- **Size**: 1.9GB
- **Should be gitignored**: Regenerated by Unity
- **Contains**: PackageCache (1.4GB), build caches, compiled assets
- **Recommendation**: Add to .gitignore to reduce repository size

### Meta Files
- Unity automatically generates .meta files
- Required for proper asset tracking
- Should be committed to version control
- **Count**: 13,944 files
