1) Soft Glow Button

button + hover
<button class="btn">Hover Me</button>

.btn{
  padding:10px 16px;
  border-radius:12px;
  border:1px solid #2a335f;
  background:#182048;
  color:#eaf7ff;
  box-shadow:0 0 0 0 rgba(124,240,255,.0);
  transition:box-shadow .25s, transform .15s;
}
.btn:hover{ 
  box-shadow:0 0 0 6px rgba(124,240,255,.15), 0 6px 18px rgba(0,0,0,.35);
  transform:translateY(-1px);
}
Tweak border-radius for shape, change box-shadow blur/spread for glow size, and adjust transform for lift amount.

7) Tooltip

data-attr + ::after
<button class="tip" data-tip="I am a tooltip!">Hover</button>

.tip{position:relative; padding:8px 12px; border-radius:10px; border:1px solid #2a335f; background:#182048}
.tip::after{
  content:attr(data-tip);
  position:absolute; left:50%; transform:translateX(-50%) translateY(6px);
  bottom:100%;
  white-space:nowrap; padding:6px 8px; border-radius:8px;
  background:#0e1330; border:1px solid #2a335f; opacity:0; pointer-events:none;
  transition:.2s; font-size:12px;
}
.tip:hover::after{opacity:1; transform:translateX(-50%) translateY(2px)}
Edit text via data-tip. Move placement by adjusting bottom/top and the transform.

12) Theme Variables

:root vars
Chip
Alt
/* Base */
:root{ --chip-bg:#182048; --chip-ink:#eaf7ff; --chip-bd:#2a335f }
.chip{ display:inline-block; padding:6px 10px; border:1px solid var(--chip-bd);
       background:var(--chip-bg); color:var(--chip-ink); border-radius:999px }
/* Alternate theme */
:root.alt{ --chip-bg:#1f2b13; --chip-ink:#f2ffe0; --chip-bd:#3f5a2a }
/* Toggle by adding .alt on <html> or <body> */
Define a palette as CSS variables in :root. Flip themes by toggling a class that overrides those variables.
