# errl-fluid

A React-based fluid simulation and visualization application using Three.js and React Three Fiber.

## Overview

errl-fluid is a web application for creating and visualizing fluid simulations with interactive controls and real-time rendering.

## Features

- Fluid simulation and visualization
- React Three Fiber integration
- Interactive controls
- Real-time rendering
- Physics-based animations

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
