# Figma Clone Engine

A high-performance, component-based design tool built with React, TypeScript, and HTML5 Canvas.

## Features
- **Infinite Canvas:** Pan, Zoom, and Grid with snap-to-grid support
- **Component Architecture:** Master/Instance inheritance model with drag-to-canvas instance creation
- **Auto-Layout:** Flexbox-based constraint solver for automated spacing
- **Vector Tools:** Polyline drawing, shapes (Line, Arrow, Ellipse, Polygon, Star)
- **Design Tools:** Full suite of 17+ tools with pop-out menus
- **Export:** PNG, SVG, and JPG export with multiple scale options
- **Eyedropper Tool:** Color picking directly from canvas (I key)
- **Context Menu:** Right-click menu for quick access to common operations
- **Layer Operations:** Z-index management, Lock/Unlock, Hide/Show
- **Page Management:** Add and delete pages with inline editing
- **Alignment Guides:** Visual guides when aligning objects
- **Rulers:** Measurement rulers with zoom-aware tick marks
- **Comments:** Add comment bubbles to collaborate
- **Developer Handoff:** Instant CSS/React code generation and smart measurements
- **Persistence:** Local JSON file save/load
- **Resizable Panels:** Customizable left and right panel widths
- **File Menu:** Comprehensive menu system with all standard operations

## Getting Started

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Run the dev server:**
    ```bash
    npm run dev
    ```

3.  **Open your browser** to the URL shown (usually `http://localhost:5173`).
