# Quick Start Guide

## Navigation
| Action | Shortcut |
| :--- | :--- |
| **Pan Canvas** | Hold `Space` + Drag / Middle Mouse |
| **Zoom** | `Ctrl` + Scroll / `Cmd` + Scroll |
| **Undo** | `Ctrl` + `Z` |
| **Redo** | `Ctrl` + `Y` |

## Tools
| Tool | Shortcut | Description |
| :--- | :--- | :--- |
| **Select** | `V` | Move and Resize objects. Click chevron for Move, Hand, Scale. |
| **Frame** | `F` | Create container frames. Click chevron for Section, Frame, Slice. |
| **Rectangle** | `R` | Draw boxes. Click chevron for Rectangle, Line, Arrow, Ellipse, Polygon, Star, Image. |
| **Text** | `T` | Click to add text. Double-click to edit. |
| **Pen** | `P` | Click to plot vector points. Click chevron for Pen, Pencil. |
| **Hand** | `H` | Pan the view without selecting. |
| **Scale** | `K` | Scale selected objects from center. |
| **Line** | `L` | Draw straight lines. |
| **Arrow** | `⇧L` | Draw arrows. |
| **Ellipse** | `O` | Draw circles and ellipses. |
| **Section** | `⇧S` | Create section frames. |
| **Slice** | `S` | Create slice frames for export. |
| **Pencil** | `⇧P` | Draw freehand paths. |
| **Comment** | Click icon | Add comment bubbles to canvas. |

## Organization
- **Group Selection:** Select multiple items and press the **Group** icon in the toolbar.
- **Create Component:** Select a Frame and click the **Diamond** component icon.
- **Create Instance:** Drag a component from the **Assets** tab in the left sidebar.

## Developer Mode
1. Toggle the switcher in the top-left from **Design** to **Dev Mode**.
2. Click any element to see its generated **CSS** and **React** code.
3. Hover over adjacent elements to see pixel-perfect measurements.
