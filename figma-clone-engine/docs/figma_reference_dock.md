# Figma Dock Reference

## Tools with Chevrons (Pop-out Menus)

### 1. Move/Select Tool (V) - HAS CHEVRON
- Icon: Mouse pointer/cursor
- Pop-out menu options:
  - Move (V) - checkmark icon, currently selected
  - Hand tool (H) - hand icon
  - Scale (K) - scale icon (two arrows pointing outwards)

### 2. Hash/Grid Tool - HAS CHEVRON
- Icon: Grid/hash symbol (#)
- Pop-out menu: Frame options
  - Section (Shift+S)
  - Frame (F) - checkmark
  - Slice (S)

### 3. Square/Rectangle Tool - HAS CHEVRON
- Icon: Square outline
- Pop-out menu: Shape options
  - Rectangle (R) - checkmark, currently selected
  - Line (L)
  - Arrow (Shift+L)
  - Ellipse (O)
  - Polygon
  - Star
  - Image/video... (Shift+Cmd+K)

## Tools WITHOUT Chevrons

### 5. Text Tool (T)
- Icon: Capital "T"
- No chevron, no pop-out menu

### 6. Circle/Ellipse Tool
- Icon: Circle outline
- No chevron, no pop-out menu

### 7. Components Tool
- Icon: Four small squares with plus sign
- No chevron, no pop-out menu

### 8. Squiggly Line Tool
- Icon: Wavy/squiggly line
- No chevron, no pop-out menu

### 9. Ruler/Measure Tool
- Icon: Ruler with cursor
- No chevron, no pop-out menu

### 10. Code Tool (</>)
- Icon: Angle brackets
- No chevron, no pop-out menu

## Visual Style
- Dock: White rounded rectangle with subtle gray border
- Active tool: Blue square background with white icon
- Inactive tools: White outline icons on dark gray background
- Chevrons: Small downward-pointing chevrons, positioned to the right of icons
- Pop-out menus: Dark gray rounded rectangles with white text, appear above the tool
- Menu items: Icon + text + keyboard shortcut on right
- Selected menu item: Blue highlight with checkmark
