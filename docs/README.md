# Documentation Index

**Location:** `docs/`  
**Purpose:** Central documentation for the Projects workspace

## Documentation Structure

This directory contains all workspace-level documentation organized by category.

### [Consolidation Documentation](consolidation/)
All consolidation-related documentation including strategy, roadmap, implementation notes, and verification reports.

- [Consolidation Index](consolidation/README.md)
- Includes: CONSOLIDATION_STRATEGY, CONSOLIDATION_ROADMAP, CONSOLIDATION_MASTER_SUMMARY, and related files

### [Cleanup Documentation](cleanup/)
Documentation related to cleanup and archive operations.

- [Cleanup Index](cleanup/README.md)
- Includes: Cleanup summaries, archive organization plans, and file management documentation

### [Phase Documentation](phases/)
Documentation organized by development phases (Phase 1, Phase 2, etc.).

- [Phase Index](phases/README.md)
- Includes: Build verification, code quality reviews, test coverage, feature verification, and production readiness

### [Status Reports](status/)
Status reports, verification documents, and completion summaries.

- [Status Index](status/README.md)
- Includes: Project status, verification reports, completion summaries, build/test status

### [Planning Documents](planning/)
Planning documents, action plans, and strategic planning.

- [Planning Index](planning/README.md)
- Includes: Action plans, extraction plans, strategy documents, and estimates

### [Guides and Standards](guides/)
Integration guides, standards, and how-to documentation.

- [Guides Index](guides/README.md)
- Includes: Component integration guides, documentation standards, technical guides, and troubleshooting

### [Analysis and Reviews](analysis/)
Analysis reports, reviews, and assessment documentation.

- [Analysis Index](analysis/README.md)
- Includes: Project analysis, code quality reviews, design analysis, and dependency mapping

### [Migration Guides](migration-guides/)
Step-by-step guides for migrating projects to use shared utilities.

- [Migration Guides Index](migration-guides/README.md)
- Includes: Component libraries, design systems, export utilities, history hooks, liquid light shows, Paper.js

### [Architecture Decision Records](decisions/)
Architecture decision records (ADRs) for consolidation work and major decisions.

- Includes: Design system consolidation, shared history hook, liquid light show merge, Three.js utilities, FX tools decision
