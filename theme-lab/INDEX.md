## Project Summary

Theme Lab is a powerful design system playground for previewing, testing, and exporting theme tokens and UI components. It provides 25 pre-built themes with a complete set of design tokens, all previewable in one interactive page.

**Key Features:**
- 25 pre-built themes (from "Errl Core" to "Mono Minimal")
- Live theme preview with instant switching
- Layout controls (density, borders, shadows)
- Complete component library (buttons, inputs, toggles, modals, typography)
- Export capabilities (JSON, CSS, code snippets)
- TypeScript for type safety
- Fully responsive design

**Theme System:**
- CSS custom properties for all theming
- Color tokens (bg, surface, accent, text, etc.)
- Gradient tokens
- Layout tokens (density, spacing, etc.)
- Shape & timing tokens (radius, shadows, transitions)
