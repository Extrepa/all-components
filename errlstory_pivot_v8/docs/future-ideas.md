## Flash Nostalgia Features

### Phase 1: Polish & Feel (High Priority)
**Estimated Effort**: Medium
**Impact**: High

#### Wiggling Crit Numbers
- Enhance `FloatingTextManager` with bounce/wiggle animation
- Yellow, bouncy font that hops upward
- Slightly larger size for crits
- Flash-style visual feedback

#### Hitstop on Big Hits
- Freeze frames briefly on crits or boss hits
- 1-3 frame pause for "crunchy" feel
- Cheap timing trick for impact feedback

#### Screen Vignette on Low HP
- Red vignette when HP < 30%
- Subtle visual feedback
- Flash drama mode

#### Sarcastic Item Descriptions
- Witty, humorous item descriptions
- "Slime Jelly – slightly illegal in 12 provinces"
- Easy to add to existing item system

### Phase 2: Systems (Medium Priority)
**Estimated Effort**: Medium
**Impact**: Medium

#### Loot Rarity System
- Color-coded items: White (Common) → Green (Uncommon) → Blue (Rare) → Purple (Epic)
- Visual rarity flashes on drop
- Fits existing item system perfectly

#### Flash-Style Achievement Popups
- Top-right corner popups
- "Ting!" sound effect
- "Achievement Unlocked: You Stepped On A Slime"
- Lightweight system

#### Enhanced Level-Up Animation
- Bigger, flashier level-up effect
- Massive "LEVEL UP!" text
- Sparkles, screen flash
- Trumpet fanfare sound

### Phase 3: Gameplay (Low Priority)
**Estimated Effort**: High
**Impact**: Medium

#### Wall Sliding
- Slide down walls when holding against them
- Like buttered toast
- Adds movement depth

#### Bounce Pads
- Glowing platforms that launch player
- Big glowy mushrooms
- Classic Flash platformer element

#### Charge Attacks
- Hold X to charge magic
- Glow effect while charging
- Release for empowered attack
- "Hold → glow → RELEASE"
