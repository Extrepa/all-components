# Hardware Setup

## Required Hardware

- Dell OptiPlex 7050
- 2TB 2.5" SATA SSD (data drive)
  - Recommended models:
    - Samsung 870 EVO — 2TB
    - Crucial MX500 — 2TB
- USB flash drive (8GB or larger) for Ubuntu installer
- Ethernet cable (server must be wired to router)

## Optional Hardware

- NVMe M.2 SSD (500GB–1TB) for OS
  - Only needed if Dell does not already have an SSD
  - Recommended:
    - Samsung 970 / 980
    - WD Blue SN570 / SN580
    - Crucial P3 / P5
- UPS (Battery Backup) for power protection and graceful shutdown

## Tools Needed

- Small Phillips screwdriver
- Clean table or desk

## Installation Order

1. Shut down Dell completely
2. Unplug all cables
3. Lay Dell on its side
4. Release side panel latch / screw
5. Remove side panel
6. Install 2TB SATA SSD:
   - Locate 2.5" drive bay/caddy
   - Slide SSD into caddy
   - Connect SATA data cable → motherboard
   - Connect SATA power cable → PSU
   - Secure drive in place
7. (Optional) Install NVMe SSD:
   - Locate M.2 slot on motherboard
   - Insert NVMe at angle
   - Screw it down gently
8. Close side panel
9. Reconnect power + monitor + keyboard + ethernet

**STOP HERE if SSDs are not installed yet.**

Do not proceed without confirming drives are seated properly.
