# Public Website

## Purpose

- Landing page
- Gallery previews
- Embedded media players
- Contact links

## Hosting

- Cheap shared hosting
- Static files preferred
- Served via Caddy on home server (or separate host)

## Rules

- Never store master files
- Never edit directly on host
- Everything syncs from server
- Public content always copied, never moved

The public site is disposable.
Can be deleted and rebuilt anytime.
