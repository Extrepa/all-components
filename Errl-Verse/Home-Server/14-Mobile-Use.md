# Mobile Usage Guide

This system is designed to be mobile-first.

## What Works on Mobile

- File uploads
- Media streaming (Jellyfin)
- Photo galleries
- Video playback
- Book reading

## What Does NOT Happen on Mobile

- Server configuration
- Docker management
- System updates

Mobile devices are for consumption and light uploads.
Administration stays on desktop.

## Key Rule

If something is annoying on mobile, simplify the UI.
Do not add features.
