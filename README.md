## Directory Structure

### errl-club-ui/
The largest collection with 50+ UI components including:
- Base components: Button, Dropdown, InputField, Modal, Slider, VibesLiquidBar, VibesMarquee
- UI modules: AssetManagerUI, AudioPlayer, CameraSettingsUI, ControlDock, etc.
- Screen components: FriendsList, MainMenu, ProfileScreen, RoomBrowser, SettingsScreen

### figma-clone-engine/
41 React/TypeScript components including:
- Main components: LayersPanel, InspectorPanel, Rulers, AlignmentGuides
- Inspector system: controls, sections, hooks, and utilities

### errl_vibecheck/
11 components including:
- App, Collection, FeedItem, Header, Renderer, Result, Screensaver, etc.

### errl-forge/
12 asset remixer components:
- AssetEditor, AssetLibrary, AnimationGenerator, BatchGenerator, etc.

### errl-portal/
5 UI components (button, card, input, scroll-area, tabs)

### errl-portal-shared/
Shared components including:
- Component library catalog

- SVG utilities
- UI components

### Errl_Components/
13 components including:
- BubbleButton, BubbleMesh, ErrlContentLayout, TrippyScene, etc.
- Shaders and stores

### errl_scene_builder/
24 components including:
- AppShell, AssetPanel, CanvasSettings, ErrlEditor, SceneViewport, etc.

## Preview System

### Quick Start

1. **Open preview.html** in your browser to browse all components
2. **For React previews:** Start the preview system:
   ```bash
   cd preview
   npm install
   npm run dev
   ```

### Features

- **Live Previews** - Click any component card to see preview
- **Syntax Highlighting** - Code previews with syntax highlighting
- **Dark Mode** - Toggle between light and dark themes
- **Advanced Filtering** - Filter by project, type, search
- **Performance Optimized** - Caching, lazy loading, concurrent limits
- **All 205 Components** - Complete coverage with type-specific handling
