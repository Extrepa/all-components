## Quick Start Testing

### 1. Enable Plugin
1. Open Obsidian
2. Settings → Community plugins
3. Enable "Errl OS"
4. Check console for "Loading Errl OS plugin" message

### 2. First-Run Wizard
- Should appear automatically
- If not, delete `data.json` in plugin directory and reload
- Complete all 5 steps
- Verify buttons visible and functional

### 3. Core Features
- **Dashboard:** Should open automatically (if enabled)
- **Capture:** Press `Ctrl/Cmd + Shift + C`
- **Commands:** Check Command Palette for "Errl:" commands

### 4. Settings
- Go to Settings → Errl OS
- Verify all settings accessible
- Test organ enable/disable
- Test path configuration
