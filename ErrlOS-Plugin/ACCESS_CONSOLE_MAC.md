# How to Access Console on Mac Obsidian

## Method 1: Keyboard Shortcut (Easiest)

Press: **`Cmd + Option + I`**

This should open the Developer Tools/Console directly.

## Method 2: Command Palette

1. Press **`Cmd + P`** (Command Palette)
2. Type: **"Developer"** or **"Console"**
3. Look for:
   - "Open Developer Tools"
   - "Toggle Developer Tools"
   - "Show Console"

## Method 3: Settings Menu

1. Go to **Settings** (gear icon or `Cmd + ,`)
2. Look for:
   - **Advanced** section
   - **Developer** section
   - Or search for "developer" in settings

## Method 4: View Menu

1. Look at the top menu bar
2. Click **View**
3. Look for:
   - "Developer Tools"
   - "Toggle Developer Tools"
   - "Show Console"

## Method 5: Right-Click Menu

1. Right-click anywhere in Obsidian
2. Look for "Inspect" or "Developer Tools" option

## Once Console is Open

1. Click the **Console** tab at the top
2. Look for error messages (usually in red)
3. Look for messages starting with `[Errl OS]`
4. Copy any error messages you see

## Alternative: Check Plugin Status

If you can't access console, try this:

1. Go to **Settings → Community plugins**
2. Find "Errl OS" in the list
3. Check if there's an error message next to it
4. Try toggling it OFF and ON again

## Still Can't Find It?

The error might also appear in:
- Settings → Community plugins → (click the "i" or info icon next to Errl OS)
- The plugin list might show a red error indicator
- Try restarting Obsidian and checking again
