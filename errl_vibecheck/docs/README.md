## Quick Overview

**VibeCheck** is an AI-powered visual coding playground that allows users to quickly batch test prompts with visual outputs. It uses Google's Gemini AI models to generate code in various formats (P5.js, SVG, HTML, GLSL shaders, Three.js, etc.) and displays them in an interactive interface.
