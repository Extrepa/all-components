# Errl OS Alpha ‚Äì FX Lab & Animation Forge

**Codename:** Errl OS ‚Äì Alpha  
**Version:** `v0.1.0-alpha`  
**Core File:** `index.html` (single-file app, lives in `/app/`)  

Errl OS Alpha is a self-contained, browser-based workstation for turning an Errl SVG into:

- Live FX labs (p5.js, HTML FX, voxels, 3D wireframes)
- Audio-reactive animations

# Errl OS Alpha (FX Lab & Animation Forge)

Errl OS Alpha is a single-file web app that turns an Errl SVG into:
- Audio-reactive FX labs (p5 / HTML / 3D / voxel)
- Captured frame galleries + favorites + tags
- Spritesheets, timelines, and game-ready animation packs
- Project profiles, session snapshots, AI handoff prompts, and a command palette

1. Clone or download this repo.
2. Open `app/index.html` in a modern browser.
   - Or run `python -m http.server 8000` inside `app/` and visit `http://localhost:8000`.
3. Load your Errl SVG and start experimenting.
