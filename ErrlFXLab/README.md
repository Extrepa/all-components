# ErrlFXLab

A web application for creative coding and visual effects experimentation.

## Overview

ErrlFXLab is a web-based creative coding laboratory for experimenting with visual effects, graphics, and interactive visualizations.

## Features

- Creative coding tools
- Visual effects experimentation
- Interactive graphics
- Modular refactoring system

## Quick Start

Open `index.html` in a web browser to launch the application.
